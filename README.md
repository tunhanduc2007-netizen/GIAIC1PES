# Champions League PES 2021 Tournament Manager

Website quản lý giải đấu PES 2021 phong cách Champions League hiện đại, đẹp, responsive.

## 🚀 Tính năng chính

- **Dashboard**: Tổng quan số liệu, BXH rút gọn và các trận đấu gần nhất.
- **Quản lý người chơi**: Thêm, xóa, sửa thông tin người chơi và đội bóng yêu thích.
- **Bảng xếp hạng tự động**: Tính điểm (+3 thắng, +1 hòa), hiệu số và sắp xếp tự động.
- **Nhập kết quả**: Ghi lại tỉ số các trận đấu và chọn MVP.
- **Vòng quay ngẫu nhiên**: Chia đội A/B với hiệu ứng vòng quay gaming.
- **Phần thưởng**: Hiển thị Podium Top 3 với hiệu ứng pháo hoa chúc mừng.
- **Import/Export Excel**: Lưu trữ và tải dữ liệu từ file Excel.
- **Bảng thủ công**: Tự tạo thêm các loại bảng dữ liệu khác tùy ý.

## 🛠 Công nghệ sử dụng

- **ReactJS** + **Vite**
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **SheetJS** (Excel processing)
- **Canvas Confetti** (Celebrations)
- **LocalStorage** (Data persistence)

## 📦 Hướng dẫn cài đặt

1. Giải nén thư mục dự án.
2. Mở terminal tại thư mục dự án.
3. Cài đặt dependencies:
   ```bash
   npm install
   ```
4. Chạy dự án ở chế độ phát triển:
   ```bash
   npm start
   ```
5. Mở trình duyệt theo địa chỉ `http://localhost:5173`.

## 🎨 Giao diện

- **Tone màu**: Champions League (Dark Blue, Neon Cyan, Star Blue).
- **Phong cách**: Gaming UI, Glassmorphism, Neon Glow.
- **Responsive**: Hỗ trợ tốt trên cả máy tính và thiết bị di động.
